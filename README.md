Advanced molecular annotation and entity resolution system for biological and chemical research.

## Features
* **Molecular Annotation**: Precision resolution of chemical and biological entities.
* **Ontology Integration**: Support for Gene Ontology, ChEBI, and MeSH.
* **High-Performance Processing**: Parallel batch processing for large datasets.
* **Comprehensive Export**: Detailed CSV reports with cross-database identifiers and direct links.

## Usage
1. Configure your API credentials in the settings tab.
2. Upload or paste your list of entities.
3. Run the analysis pipeline to retrieve annotations.
4. Export results for further research.
